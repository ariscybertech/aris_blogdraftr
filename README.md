# Blog Drafter: AWS Amplify & Appcircle Demo App

## Getting Started

To build the app, follow these steps:

1. Accept invite to Blog Drafter Amplify app email 
2. Download the repo to your machine
3. Open the terminal to the root directory of the app
4. Run the `$ amplify pull`
5. Select Blog Drafter from the CLI prompt
6. Open the Flutter app
7. If using VSCode, navigate to the `pubspec.yaml` file and tap the **Get Packages** button at the top right. If not, run `$ flutter pub get` from the terminal at the root directory of the project.
8. To run the project, run `$ flutter run --no-sound-null-safety`
